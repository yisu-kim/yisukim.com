# yisukim.com

## Getting started

```bash
npm i
npm run dev
```

## Stacks

- TypeScript
- Next.js 14 (app router)
- React.js 18
- Shadcn UI (Radix UI + Tailwind CSS)
- MDX (remark and rehype plugins)
- Vercel

## Features

- [x] multilingual support
- [x] MDX support
- [x] code highlighting
- [x] responsive design
- [x] dark mode
- [x] SEO (sitemap, open graph configuration, metadata, robots.txt)
- [x] GA
- [x] SSG
- [x] custom domain
- [x] not found page
